# atmosy

**Atmosy App**

Atmosy is a production-ready weather forecasting application built with Flutter, following the MVVM architecture and using BLoC for state management. The app provides a modern, responsive, and user-friendly interface with short background videos and weather images synchronized to current weather conditions.


## Key Features


Home Screen: Displays the current weather and a 7-day forecast.

City Search: Search for weather in any city worldwide.


Dark Mode Support: Switch between light and dark themes.

Error Handling: Handles invalid city names, internet disconnection, and other errors gracefully.

Repository Pattern: Fetches weather data from the OpenWeatherMap API.

Model Classes: Implemented using Freezed or Equatable for immutability and equality checks.

Organized Folder Structure: Includes folders for models, repository, ViewModel/BLoC, and UI components.

Responsive UI: Adapts to different screen sizes and devices.

Core Utilities: Contains API client, constants, and helper functions.

Loading & Error States: Provides feedback to users when loading or errors occur.
